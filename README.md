# jamstack-api
API for a JAMstack tutorial!

## POST
Need 'jam' and 'msrp' keys in the body of the request (x-www-form-urlencoded)

## GET
Need the ID of the resource in the URI

## PUT
Need the ID of the resource in the URI
Need 'jam' and 'msrp' keys in the body of the request (x-www-form-urlencoded)

## DELETE
Need the ID of the resource in the URI
